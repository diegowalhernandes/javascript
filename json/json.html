O que é Json?

  • JSON = JavaScript Object Notation;
  • Um formato de representação de dados;
  • Mais simples que o XML, que é utilizado para fins parecidos;
  • Utiliza o formato chave e valor;
  • É leve para ser enviado por requisições;
  • Muito utilizado para API e também arquivos de configuração;

    • Tipos de dados

  • O json aceita diversos tipos de dados;
  • Strings - "Olá Mundo!"
  • Números - 1 ou 136.369;
  • Arrays - [1,2,3];
  • Objetos - {"Nome": "Diego"};
  • Dados nulos - null;
